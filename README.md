# Proyecto: Bootstrap 5 Showcase

Este proyecto es una página web responsiva que implementa todos los componentes y variantes de formularios de Bootstrap 5, siguiendo buenas prácticas de desarrollo.

## Objetivo

Demostrar el uso correcto de la biblioteca Bootstrap 5.0, manteniendo una estructura de proyecto ordenada (separación de responsabilidades) y código semántico.

## Estructura del Proyecto

## Tecnologías utilizadas
- **HTML** – Estructura y contenido.
- **CSS** – Estilos personalizados y responsivos.
- **JavaScript** – Validación de formularios, toasts, modals, tooltips y popovers.
- **Bootstrap 5** – Componentes, diseño responsivo e interactividad.

## Funcionalidades principales
- **Navbar**: Con links internos, dropdown y menú responsive.
- **Carousel**: Muestra imágenes representativas del curso.
- **Sección de contenidos**: Cards, botones, badges, accordion, progress bar, spinner.
- **Formularios**: Inputs de texto, email, fecha, URL, radio, checkbox, switch, select, file, floating labels y rangos.
- **Interactividad**:
  - Modal informativo sobre el curso.
  - Toast para notificaciones rápidas.
  - Offcanvas lateral con descripción del curso.
  - Tabs con Temas y Horarios.
- **Listas y Paginación**: Muestra módulos y organización del contenido.
- **Responsive**: Diseño adaptado para móviles, tablets y escritorio.

## Instrucciones de uso
1. Clonar el repositorio o descargar los archivos.
2. Mantener la estructura de carpetas tal como está (`público/vendor`, `público/css`, `público/js`, `públic/imagenes`).
3. Abrir `index.html` en un navegador moderno.
4. Editar `estilos.css` para personalización visual.
5. Editar `main.js` para agregar o modificar funciones interactivas.
